## 1. 欢迎，指挥官

**坦克工程师** 是一款几何物理坦克对战游戏。核心循环很简单：

```
搜集零件 → 组装坦克 → 进入对战 → 获得奖励 → 解锁更多零件
```

你操控一辆自定义坦克，在基于真实物理规则的战场上与敌人对决。子弹会反弹、方块会被推动、墙壁会被摧毁——每一次射击都是一道几何题。

打开游戏后，你会看到一个动态的 AI 对战封面。点击 **START GAME** 进入对战大厅。

---

## 2. 第一步：组装你的坦克

在对战大厅底部，点击 **🔧 组装** 进入车间。

### 初次进入

车间顶部有三个配置槽：**配置1**、**配置2**、**配置3**。默认选中配置1（绿色高亮）。

左侧面板有四个标签页：
- 🔫 **炮管** — 决定子弹怎么打
- 🛡️ **炮塔** — 决定能扛多少
- 🏎️ **车身** — 决定怎么移动
- 🎖️ **车长** — 决定关键时刻能做什么

### 选择零件

点击任意零件卡片即可装备。**选中的零件自动保存到当前配置槽**，无需手动保存。

右侧预览窗口会实时显示坦克外观和属性数值。

### 快速测试

组装完成后，点击右侧 **🎯 演习** 按钮进入训练场。这里有固定靶和活动靶供你测试武器效果。按 `E` 释放车长技能。

测试完毕点击 **⏹ 结束** 返回车间，再点 **← 返回大厅**。

## 3. 理解零件系统

坦克由四种零件组成，每种决定不同属性：

### 炮管 —— "怎么打"

| 炮管 | 稀有度 | 特点 |
|------|--------|------|
| 直射管 | 普通 | 标准子弹，碰墙反弹2次。新手首选 |
| 反射管 | 稀有 | 镜面反弹，每次反弹伤害衰减20%。适合墙壁多的地图 |
| 透射管 | 史诗 | 穿透砖墙1层。攻击墙后敌人 |
| 曲射管 | 史诗 | 抛物线飞越障碍。最高点伤害×2 |
| 烟花管 | 传说 | 母弹飞行中每0.28s向6方向发射子子弹 |
| 狙击管 | 传说 | 伤害999一击必杀，穿透所有墙体 |
| 加特林 | 史诗 | 射速70ms弹幕倾泻，每发伤害6 |
| 火箭炮 | 传说 | 追踪鼠标，撞击爆炸+火圈AoE |
| 散射管 | 传说 | 一次射出3颗15°扇形子弹 |
| 磁轨管 | 传说 | 子弹沿金属墙滑行，适合金属墙密集地图 |

### 炮塔 —— "能扛多少"

| 炮塔 | 特点 |
|------|------|
| 轻甲 | HP 80，重量轻，速度加成 |
| 重甲 | HP 180，减伤15%。适合正面硬抗 |
| 反应装甲 | 受击触发0.5s无敌（冷却8s）。高手向 |
| 修复装甲 | 脱战3s后每秒回5HP |
| 镜像装甲 | 30%概率反弹伤害给攻击者 |
| 狂战装甲 | HP越低伤害越高（最多×2） |

### 车身 —— "怎么移动"

| 车身 | 特点 |
|------|------|
| 标准 | 均衡无特殊效果 |
| 惯性 | 松手后惯性滑行，速度1.2× |
| 重型 | 速度0.7×但能碾碎砖墙 |
| 履带 | 瞬间转向（无角速度延迟） |
| 闪现 | Shift瞬移3格 |
| 冲刺 | 持续移动时速度逐步提升至1.5× |
| 悬浮 | 全地形滑行（类似冰面），速度1.1× |

### 重量等级

每个零件有重量（1-3）。总重量决定坦克等级：

| 等级 | 重量 | 速度 | 效果 |
|------|------|------|------|
| 🪶 轻量级 | ≤4 | +15% | 后坐力后退1格 |
| ⚖️ 中量级 | 5-6 | ±0% | 无特殊 |
| 🏋️ 重量级 | ≥7 | -15% | 免疫后坐力，碾碎砖墙 |

**策略提示**：你不能同时带传说炮管+传说炮塔+传说车身——总重量会超过限制，让你变成乌龟。

## 4. 你的第一场战斗：围城模式

### 什么是围城

地图中央有一座指挥所（500 HP）。敌人从四面八方涌来，持续3分钟。你的任务：**活下去，保护指挥所**。

### 操作基础

| 操作 | 按键 |
|------|------|
| 移动 | `W` `A` `S` `D` |
| 炮塔瞄准 | 鼠标移动 |
| 开火 | 鼠标左键 或 `Space` |
| 释放技能 | `E` |
| 调试视野 | `U` |
| 召唤Boss | `O`（测试用） |

### 战斗要诀

1. **利用墙壁**：砖墙可被摧毁（500 HP），金属墙不可摧毁但可反弹子弹。用反射管时，站好角度，让子弹在两面墙之间弹射命中敌人。

2. **保护指挥所**：敌人会优先攻击指挥所。不要离指挥所太远——它也是你的最后防线。

3. **注意波次提示**：每30秒一波新敌人。屏幕顶部显示计时器和当前波次。

4. **善用技能**：车长技能是你的底牌。重力井可以聚怪、闪电可以清场、维修可以救命。

5. **拿好奖励**：存活3分钟=胜利，获得大量金币+稀有零件。

## 5. 车长技能：关键时刻的王牌

车长是坦克的第四维零件。每个车长有一个主动技能（按 `E` 释放），冷却时间各不相同。

### 新手推荐

| 车长 | 技能 | 冷却 | 推荐理由 |
|------|------|------|---------|
| 维修专家 | 立即恢复40HP | 20s | 最简单的自保技能，适合新手 |
| 冲刺指挥官 | 2s内移速翻倍 | 20s | 快速脱离包围或追击残血 |
| 闪电指挥官 | 连锁闪电×5，每击100 | 20s | AOE清场，适合敌人密集时 |
| 炮塔工程师 | 部署自动炮塔 | 20s | 额外火力点，协同重甲回血 |

### 进阶技能

当你熟悉基础操作后，试试这些传说级技能：

- **三体人**（☄️）：鼠标位置召唤陨石。2秒锁定→火球砸落→半径360清场+火海
- **歌者**（📐）：二向箔压缩。画面压扁成一条线，全清敌人+方块+砖墙
- **奥德修斯**（🏛️）：特洛伊木马。巨马从左侧滑入→开门释放6辆友军坦克
- **诺亚**（🌊）：大洪水23秒。涨潮→峰值→退潮，俘虏一切
- **雷神**（⚡）：Mjolnir锤子。飞出后绕玩家轨道飞行，撞飞敌人

## 6. 双王战争：三路推进

### 模式介绍

横版对称地图，蓝方（你）vs 红方（AI）。中轴河流+三座桥分隔。双方各有1主基地+3防御塔。

### 核心规则

1. **自动出兵**：每10秒双方各生成3辆AI坦克（每路1辆），10秒一波
2. **三路推进**：上中下三条进攻路线，AI沿航线自动进攻
3. **攻击优先级**：敌军坦克 > 防御塔 > 主基地
4. **胜利条件**：摧毁敌方主基地 / 3分钟内己方基地未被摧毁

### 策略

- **保护你的防御塔**：塔被摧毁后那条路就失去自动防御。优先攻击敌方同路塔。
- **利用复活**：在双王中玩家死亡后会自动复活。不要怕死，但要保护基地。
- **基地回血**：站在蓝方基地附近会自动回血（绿色粒子+绿色虚线圆圈）。
- **观察航线**：地图上有三条白色虚线标注进攻路线。AI沿这些路线推进。

## 7. 多坦克模式：三辆齐发

### 开启方式

在车间底部，点击 **👤 单坦** 切换为 **👥 多坦**。配置1/2/3 各对应一辆坦克。

### 操控方式

进入对战后：
- 按 `1` `2` `3` 切换操控坦克
- 当前操控的坦克下方有 **蓝色圆圈+编号**
- 待命坦克下方有 **白色圆圈+编号**
- 待命坦克会自动跟随你、攻击敌人

### 战术

- **分工明确**：一辆坦克装狙击管（一击必杀），一辆装散射管（中距离压制），一辆装火箭炮（AoE清场）
- **保护待命坦克**：它们是你重要的火力支援，不要离它们太远

## 8. 进阶技巧

### 反弹射击

反射管的子弹在碰到墙壁时以入射角=反射角弹开。**二次反弹命中敌人**会产生 "BOUNCE ×1.5" 伤害加成。在迷宫和交叉火力地图中，反弹射击是核心战术。

### 物块推动

射击砖块可以将它们推出去。推出去的砖块撞击敌人造成额外伤害——"BLOCK SMASH ×2"。如果一块砖块推出后又撞到另一块再撞到敌人，就是 "DOMINO ×3"。

### 零件协同效应

某些零件组合有隐藏加成（查看零件描述中带颜色标注的提示）：

- **烟花管 + 弹幕指挥官** → 分裂频率翻倍（"烟花祭"）
- **狙击管 + 空军上校** → 轰炸频率翻倍+十字交叉轰炸
- **火箭炮 + 巫师** → 火箭击杀自动复活为友军
- **冲刺底盘 + 忍者/闪电** → 影分身数量翻倍
- **重型底盘 + 炮塔工程师** → 炮塔范围内每秒回8HP
