## 1. 核心愿景

### 1.1 我们要做什么

一款**浏览器原生**的坦克对战游戏。玩家搜集零件、组装独一无二的坦克，在基于**真实几何物理规则**（反弹、惯性、动量）的战场上对决。每局控制在 3 分钟内，适合工作学习间隙来一把。

### 1.2 三大设计支柱

| 支柱 | 说明 |
|------|------|
| 🧩 **组装** | 坦克由炮管、炮塔、车身、车长四种零件拼成，零件决定能力，组合产生策略 |
| 📐 **几何** | 子弹反弹、惯性滑行、抛物线——数学物理是玩法的核心，不是装饰 |
| ⚡ **轻量** | 3 分钟一局，浏览器即开即玩，零安装，极简 UI |

### 1.3 设计哲学

- **极简操作，深度涌现** — 方向键 + 空格键就能玩，但零件组合 × 几何环境产生无限策略
- **数学即玩法** — 每一个零件差异背后都有数学概念（反射、抛物线、动量守恒），游戏本身就是数学直觉训练
- **跨模式收集驱动** — 所有模式共享零件库，你在围城中生存 3 分钟获得的零件，可以在棋类模式中使用

## 3. 零件系统

### 3.1 四个维度

坦克由四种零件组成，每个零件都有独立的战术意义：

| 维度 | 图标 | 决定属性 | 设计思路 |
|------|------|---------|---------|
| **炮管** Barrel | 🔫 | 子弹类型与弹道特性 | 决定"怎么打"——输出方式的质变 |
| **炮塔** Turret | 🛡️ | 生命值与装填速度 | 决定"能扛多久"——防御与节奏 |
| **车身** Chassis | 🏎️ | 移动速度与物理特性 | 决定"怎么走"——机动方式 |
| **车长** Commander | 🎖️ | 主动技能 | 决定"关键时刻能做什么" |

### 3.2 零件详细设计

#### 炮管（Barrel）— 弹道即玩法

| 名称 | 稀有度 | 效果 | 数学概念 | 视觉提示 |
|------|--------|------|---------|---------|
| 直射管 | 普通 | 标准直线子弹，射程无限 | 直线 | 白色弹丸 |
| 反射管 | 稀有 | 子弹碰墙反弹 2 次，每次反弹角度对称 | 镜面反射 | 银色弹丸 + 反弹闪光 |
| 透射管 | 稀有 | 子弹穿透砖墙 1 层（穿过后伤害减半） | 穿透性 | 蓝色弹丸 + 穿透波纹 |
| 分裂管 | 史诗 | 子弹击中障碍后分裂为 2 颗，各偏转 45° | 对称分解 | 绿色弹丸 + 分裂粒子 |
| 曲射管 | 史诗 | 子弹走抛物线轨迹，可越过砖墙 | 二次函数 / 抛物线 | 橙色弹丸 + 弧线轨迹 |
| 磁轨管 | 传说 | 子弹自动吸附到最近的金属墙上并沿墙滑行 | 最短路径 / 约束优化 | 紫色弹丸 + 电磁特效 |
| 散射管 | 传说 | 一次射出 3 颗呈 15° 扇形分布的子弹 | 角度分布 | 红色散射弹幕 |

#### 炮塔（Turret）— 攻防节奏

| 名称 | 稀有度 | 效果 | 重量 |
|------|--------|------|------|
| 轻甲炮塔 | 普通 | HP 80，装填时间 0.8s | 1 |
| 标准炮塔 | 普通 | HP 120，装填时间 1.0s | 2 |
| 重甲炮塔 | 稀有 | HP 180，装填时间 1.3s | 3 |
| 反应装甲 | 史诗 | HP 90，受击瞬间0.5s 无敌帧（需手动触发） | 2 |
| 修复装甲 | 稀有 | HP 100，脱离战斗 3s 后缓慢回血 | 2 |
| 镜像装甲 | 传说 | HP 60，被击中时有 30% 概率将伤害反弹给攻击者 | 1 |

#### 车身（Chassis）— 机动哲学

| 名称 | 稀有度 | 效果 | 重量 |
|------|--------|------|------|
| 标准底盘 | 普通 | 速度 1.0×，无特殊效果 | 2 |
| 惯性底盘 | 稀有 | 松开方向键后继续滑行 3 格（冰面效果），速度 1.2× | 1 |
| 轻量底盘 | 普通 | 速度 1.3×，但开火时后坐力后退 1 格 | 1 |
| 重型底盘 | 稀有 | 速度 0.7×，但能碾碎砖墙，且不受后坐力影响 | 3 |
| 履带底盘 | 史诗 | 速度 0.9×，可原地旋转（其他底盘只能 4/8 方向） | 2 |
| 闪现底盘 | 传说 | 速度 0.8×，每 12s 可向任意方向瞬移 3 格 | 2 |

#### 车长（Commander）— 关键技能

| 名称 | 稀有度 | 技能 | 冷却 |
|------|--------|------|------|
| 几何直觉 | 普通 | 显示子弹反弹/抛物线的瞄准预览线 3s | 15s |
| 弹幕时间 | 稀有 | 3s 内装填时间降为 0.1s | 45s |
| 烟雾弹 | 普通 | 释放烟雾遮挡敌人视野 3s | 20s |
| 紧急维修 | 稀有 | 立即恢复 40 HP | 60s |
| 地雷铺设 | 史诗 | 在当前位置放置一颗地雷（敌方触碰即炸） | 25s |
| 时停领域 | 传说 | 2s 内所有敌方子弹减速 70% | 50s |

### 3.3 零件获取途径

| 途径 | 说明 | 频率 |
|------|------|------|
| 战后掉落 | 每局对战结束随机掉落 1-2 个零件（稀有度受模式难度影响） | 每局 |
| 每日商店 | 每日刷新 3 个零件，用金币购买 | 每日 |
| 成就奖励 | 特定里程碑解锁固定传说零件 | 一次性 |
| 模式首通 | 剧情关卡首通奖励专属零件 | 一次性 |
| 赛季奖励 | PvP 排位赛季结算 | 每赛季 |

## 4. 坦克方程约束系统

### 4.1 核心公式

```
坦克总重量 W = W_炮管 + W_炮塔 + W_车身

其中：
  W_炮管 ∈ {1, 2, 3}（直射管=1，反射/透射=1，分裂/曲射=2，磁轨/散射=3）
  W_炮塔 ∈ {1, 2, 3}
  W_车身 ∈ {1, 2, 3}

总重量 W ∈ [3, 9]
```

### 4.2 重量等级效果

| 等级 | 重量范围 | 速度修正 | 特殊效果 |
|------|---------|---------|---------|
| 🪶 轻量级 | W ≤ 4 | +15% | 开火后坐力：后退 1 格 |
| ⚖️ 中量级 | 5 ≤ W ≤ 6 | ±0% | 无特殊效果 |
| 🏋️ 重量级 | W ≥ 7 | -15% | 免疫后坐力；触碰砖墙可碾碎（自身减速 50%） |

### 4.3 设计意图

- **防止最强零件堆砌**：你不能同时带传说炮管 + 传说炮塔 + 传说车身（总重量超限，会变乌龟）
- **取舍产生策略**：轻量坦克是玻璃大炮（速度快但被碰就残），重量坦克是移动堡垒（肉但慢）
- **数学可解释**：每个零件的"重量"值让玩家在组装时做资源分配决策——本质上是一个小型优化问题

## 5. 战斗核心：几何物理引擎

### 5.1 基础物理规则

| 规则 | 描述 |
|------|------|
| **移动** | 4 或 8 方向移动（取决于车身），网格化地图 |
| **碰撞** | 坦克不能穿过砖墙和金属墙，坦克之间不能重叠 |
| **惯性** | 惯性底盘车身释放方向键后继续滑行（沿原方向减速） |
| **后坐力** | 轻量坦克开火时向反方向位移 1 格 |
| **子弹反弹** | 反射管子弹碰墙以入射角 = 反射角弹射（可反弹 2 次） |
| **子弹穿透** | 透射管子弹可穿过砖墙但不可穿过金属墙 |
| **子弹抛物线** | 曲射管子弹以初速度 + 重力加速度运动 |
| **墙体重压** | 重型坦克触碰砖墙时摧毁该墙格 |

### 5.2 地图元素

| 元素 | 可摧毁 | 子弹交互 | 坦克交互 |
|------|--------|---------|---------|
| 🧱 砖墙 | ✅（坦克炮火 & 重型碾压） | 普通子弹：阻挡；穿透子弹：穿过 1 层 | 阻挡（重型：碾碎） |
| 🔩 金属墙 | ❌ | 阻挡所有子弹（磁轨弹沿墙滑行） | 阻挡 |
| 🌊 水域 | ❌ | 子弹可飞越 | 不可通过 |
| 🌿 草丛 | ❌ | 子弹可穿过 | 可通过，隐藏视野 |
| ❄️ 冰面 | ❌ | 正常 | 移动有惯性，所有底盘滑行 |
| 💥 爆桶 | ✅ | 击中即爆炸，范围伤害 | 触碰爆炸 |

### 5.3 伤害计算

```
基础伤害 D_base = 炮管基础伤害（通常 33-50）

穿透衰减：穿透砖墙后 D = D_base × 0.5
反弹衰减：每次反弹后 D = D_base × 0.8（鼓励反弹技巧）
分裂衰减：每颗分裂弹 D = D_base × 0.6
抛物线加成：抛物线最高点击中 D = D_base × 1.5（风险与回报）

实际伤害 = D × (1 - 炮塔防御系数)
```

## 6. 游戏模式矩阵

### 6.1 总览

| 模式 | 类型 | 玩家数 | 时长 | 核心体验 | 上线阶段 |
|------|------|--------|------|---------|---------|
| 🏰 围城 | 生存 | 1 | 3min | 孤胆守城，几何操作 | **MVP** |
| ♟️ 棋类 | 回合策略 | 1vAI | 3min | 每一步都是几何题 | Phase 2 |
| 🎨 涂地 | 区域控制 | 1vAI | 2-3min | 空间拉锯，涂色占地 | Phase 2 |
| 🤖 异步 PvP | 编程对战 | 2 | 3min | 坦克组装+AI 策略双比拼 | Phase 2 |
| 🎯 竞技场 | 死斗 | 1vAI | 1-3min | 纯几何对抗 | Phase 2 |
| 📖 剧情解密 | 关卡解谜 | 1 | 不限 | 零件机制教学+故事 | Phase 3 |
| 🌍 开放世界 | 刷怪探索 | 1 | 不限 | 泰拉瑞亚式探索收集 | Phase 3 |
| 🍗 吃鸡 | 大逃杀 | 多人 | 5-8min | 边打边捡，存活为王 | Phase 3 |
| 🎲 肉鸽 | 闯关 | 1 | 10-15min | 元气骑士式爬塔搜集 | Phase 3 |

### 6.2 MVP 模式：围城模式

**目标**：在敌人不断涌来的情况下，存活 3 分钟。

**规则**：
- 地图中央有一座指挥所（你的初始位置）
- 敌人从地图边缘 4 个方向刷新，每 30s 增加一波强度
- 指挥所有 500 HP，归零则失败
- 你可以自由移动，但指挥所被摧毁即判负
- 每消灭一个敌人获得金币 ×10
- 存活满 3 分钟 = 胜利，额外奖励稀有零件 ×1

**波次设计**：

| 波次 | 时间 | 敌人数量 | 敌人类型 |
|------|------|---------|---------|
| 1 | 0:00-0:30 | 2 | 基础坦克（直射管+轻甲+标准底盘） |
| 2 | 0:30-1:00 | 3 | 基础 + 1 反射管坦克 |
| 3 | 1:00-1:30 | 4 | 混合：反射管 + 重型坦克 |
| 4 | 1:30-2:00 | 5 | 混合 + 1 透射管坦克 |
| 5 | 2:00-2:30 | 6 | 全混合 + 1 史诗坦克 |
| 终波 | 2:30-3:00 | 8 | 弹幕地狱：全部敌人装填速度翻倍 |

### 6.3 棋类模式（Phase 2）

**目标**：在回合制规则下消灭所有敌方坦克。

**规则**：
- 小棋盘 8×8，双方各 3 辆坦克
- 每回合：移动 1 步 + 开 1 枪，或不动 + 开 2 枪
- 子弹穿透整条直线，被击中即摧毁
- 3 分钟计时器，超时按剩余血量判负

### 6.4 涂地模式（Phase 2）

**目标**：涂满地图 60% 面积，或消灭所有敌人。

**规则**：
- 坦克走过的格子染上己方颜色
- 敌方领地上移动速度 -30%
- 己方领地上装填速度 +20%
- 敌方也在涂色，形成空间拉锯

### 6.5 异步 PvP（Phase 2）

**目标**：坦克组装与 AI 策略的双重较量。

**规则**：
- 玩家 A 组装坦克 + 编写 AI 策略（行为树/条件列表）
- 上传后，玩家 B 挑战 A 的"幽灵坦克"
- 玩家 B 手动操作 vs AI 控制的 A 坦克
- 排行榜：谁的坦克被击败次数最少
- 本质上是"设计一个难题让别人来解"

## 7. 进度与经济系统

### 7.1 货币

| 货币 | 获取方式 | 用途 |
|------|---------|------|
| 🪙 金币 | 战后奖励、每日任务 | 商店购买零件 |
| 💎 零件碎片 | 战后掉落、分解重复零件 | 合成指定稀有度零件 |

### 7.2 每日商店

- 每日刷新 3 个随机零件
- 稀有度分布：普通 40%、稀有 35%、史诗 20%、传说 5%
- 价格：普通 100 金、稀有 500 金、史诗 2000 金、传说 10000 金
- 可消耗金币手动刷新（每日限 3 次）

### 7.3 零件库

- 所有获得的零件永久存入零件库
- 零件不会消耗（不像皇室战争的卡牌升级）
- 重复零件自动转为零件碎片
- 支持按维度、稀有度、重量筛选

## 8. 视觉风格

### 8.1 总体方向：矢量几何极简

- **坦克**：棱角分明的几何形状拼成（炮管=矩形、炮塔=梯形、车身=矩形+履带线）
- **墙壁**：粗线条勾勒的方块
- **子弹**：发光圆点 + 运动轨迹线
- **地图**：网格线隐约可见，整体干净利落

### 8.2 配色方案

| 用途 | 颜色 | 色值 |
|------|------|------|
| 玩家坦克 | 亮蓝 | `#4A9EFF` |
| 敌方坦克 | 橙红 | `#FF6B4A` |
| 砖墙 | 暖棕 | `#C4A47A` |
| 金属墙 | 冷灰 | `#8E9AAF` |
| 水域 | 浅蓝 | `#7EC8E3` |
| 冰面 | 淡青 | `#B8E8F0` |
| 草丛 | 暗绿 | `#5C8B5C` |
| 子弹 | 亮白/根据炮管类型变色 | — |
| 背景 | 深灰绿（战场感觉） | `#2C3038` |
| UI 面板 | 半透明黑底 | `rgba(0,0,0,0.7)` |

### 8.3 零件视觉语言

每种零件类型对应一种标志性的几何形状语言：
- **炮管**：尖锐三角/箭头元素
- **炮塔**：加厚的梯形/盾形元素
- **车身**：宽大的矩形/履带元素
- **车长**：星形/勋章元素

### 8.4 UI 设计原则

- **无血条原则**（尽可能）：坦克受损时外观变化（冒烟、碎裂、变色），而非头顶挂着血条
- **最少 HUD**：只显示计时器、金币、技能冷却图标
- **零件颜色对应**：坦克身上不同颜色的几何块对应不同零件，一眼可辨

## 9. 技术方案

### 9.1 技术栈

| 层级 | 技术 | 理由 |
|------|------|------|
| 渲染 | HTML5 Canvas 2D | 浏览器原生，零依赖，性能够用 |
| 语言 | TypeScript | 类型安全，适合零件系统的复杂类型 |
| 构建 | 无构建工具（ES Modules） | 极简目标——直接从 HTML 加载 TS？不现实，用 Vite 轻量构建 |
| 存储 | localStorage | 零件库、金币、设置持久化 |
| 网络 | WebSocket（后期 PvP） | 实时对战 |
| 物理 | 自研简单网格物理 | 不需要 Box2D，地图是网格化的 |

### 9.3 核心数据结构

```typescript
// 零件
interface Part {
  id: string;
  name: string;
  type: 'barrel' | 'turret' | 'chassis' | 'commander';
  rarity: 'common' | 'rare' | 'epic' | 'legendary';
  weight: number;
  stats: Record<string, number>;
  description: string;
  unlocked: boolean;
}

// 坦克 = 零件的组合
interface TankConfig {
  barrel: Part;
  turret: Part;
  chassis: Part;
  commander: Part;
  totalWeight: number;
  weightClass: 'light' | 'medium' | 'heavy';
}

// 子弹
interface Bullet {
  pos: Vector2;
  vel: Vector2;
  type: 'straight' | 'bounce' | 'pierce' | 'split' | 'arc' | 'magnetic';
  bouncesLeft: number;      // 反射管：剩余反弹次数
  piercesLeft: number;      // 透射管：剩余穿透层数
  damage: number;
  owner: 'player' | 'enemy';
}

// 地图网格
type TileType = 'empty' | 'brick' | 'metal' | 'water' | 'grass' | 'ice' | 'barrel';
type Tile = { type: TileType; color?: string; hp?: number };

// 坦克实体
interface TankEntity {
  pos: Vector2;
  dir: number;              // 朝向角度
  config: TankConfig;
  hp: number;
  cooldown: number;         // 装填剩余时间
  velocity: Vector2;        // 惯性速度
  isPlayer: boolean;
}
```

## 10. MVP 范围

### 10.1 MVP 包含

| 模块 | 内容 |
|------|------|
| ✅ 组装界面 | 车库 UI，选择炮管/炮塔/车身（3 维度各 2 个零件，共 6 个），坦克方程约束检查 |
| ✅ 围城模式 | 单人防守 3 分钟，6 波敌人，指挥所保护 |
| ✅ 几何物理 | 移动、碰撞、子弹反弹（反射管）、惯性（惯性底盘） |
| ✅ 敌人 AI | 基础寻路（向指挥所移动）+ 间隔开火 |
| ✅ 奖励系统 | 战后金币 + 随机零件掉落 |
| ✅ 商店 | 每日 3 个零件刷新，金币购买 |
| ✅ 零件库 | localStorage 持久化，筛选查看 |
| ✅ 矢量美术 | Canvas 代码绘制所有图形 |
| ✅ 地图元素 | 砖墙、金属墙（至少 2 种地形） |

### 10.2 MVP 不包含

| 模块 | 原因 |
|------|------|
| ❌ 车长系统 | MVP 先做 3 维度（炮管/炮塔/车身），减少复杂度 |
| ❌ 棋类/涂地/PvP/剧情 | Phase 2+ |
| ❌ 零件升级/融合 | 先做基础收集 |
| ❌ 音效/音乐 | 纯视觉先行 |
| ❌ 移动端适配 | 先桌面端 |
| ❌ 网络对战 | Phase 2 异步 PvP |

### 10.3 MVP 零件清单（8 个）

**炮管**：直射管（普通）、反射管（稀有）
**炮塔**：轻甲炮塔（普通）、重甲炮塔（稀有）
**车身**：标准底盘（普通）、惯性底盘（稀有）

（MVP 不包含车长维度，共 2×2×2 = 8 种坦克组合）
