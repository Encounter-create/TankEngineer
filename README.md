# Tank Engineer

> 搜集零件 · 组装坦克 · 几何物理对战

**Tank Engineer** 是一款浏览器原生的坦克对战游戏。搜集零件、组装独一无二的坦克，在基于真实几何物理规则的战场上对决。每局控制在 3 分钟内。

---

## 游戏特色

### 零件组装

49 种零件（11 炮管 + 6 炮塔 + 7 底盘 + 25 车长），自由组合。重量方程约束——轻量级玻璃大炮 vs 重量级移动堡垒，不是数值上的二选一，而是玩法上的根本不同。

### 几何物理

9 种弹道风格，每种有独立的物理规则：
- **直射** — 碰墙反弹 2 次
- **反射** — 镜面反弹，每次衰减 20%
- **穿透** — 穿砖墙 1 层，伤害减半
- **曲射** — 抛物线飞越障碍
- **烟花** — 母弹飞行中分裂 6 发子子弹
- **轨道** — 双弹 180° 相位环绕
- **火箭** — 追踪鼠标，撞击爆炸 + 火圈
- **散射** — 3 弹 15° 扇形散布
- **磁轨** — 沿金属墙滑行

方块可被子弹和坦克推动，多米诺连锁产生额外伤害。

### 车长技能

25 个典故驱动的技能——从荷马史诗到量子力学：

| 技能 | 出处 | 效果 |
|------|------|------|
| 📐 二向箔 | 《三体》 | 画面压扁成一条线，全场清空 |
| 🏛️ 特洛伊木马 | 荷马史诗 | 巨马滑入，侧门开放放出 6 辆友军 |
| 🌊 诺亚方舟 | 圣经 | 大洪水淹没全场 |
| ⚔️ 达摩克利斯之剑 | 希腊典故 | 悬剑 4.7 秒后秒杀全敌 |
| 🐱 薛定谔的猫 | 量子力学 | 红蓝双曝 5 秒，50% 敌人瞬间消亡 |
| 💥 大爆炸 | 宇宙学 | 万物坍缩后爆炸飞散 |
| ☄️ 陨石天降 | — | 大火球加速砸落，白屏震屏 |
| 🐉 叶公好龙 | 《庄子》 | 萌龙追猎，爱心粒子压死敌人 |

### 游戏模式

#### 🏰 围城

3 分钟防守指挥所。6 波敌人从地图边缘涌来，守卫中央指挥所不被摧毁。10 张地图可选。

#### 👑 双王战争

横版对称 3 路 PvE。蓝红双方向自动出兵，沿三路推进，攻击敌方防御塔和基地。3 分钟渐进波次。

#### 🎯 多坦克模式

车间开启后，3 辆坦克同时进场。按 `1` `2` `3` 切换操控，待命坦克自动跟随并攻击。

## 操作指南

| 按键 | 功能 |
|------|------|
| `W` `A` `S` `D` | 移动 |
| 鼠标移动 | 炮塔瞄准 |
| 鼠标左键 / `Space` | 开火 |
| `E` | 车长技能 |
| `1` `2` `3` | 切换操控坦克（多坦克模式） |
| `U` | 调试模式（显示 AI 视野/攻击范围） |
| `O` | 大厅按O快速测试双王 / 围城刷Boss |

## 项目结构

```
src/
├── main.ts              ← 状态机 + 游戏循环
├── core/                ← 底层引擎 (Physics, GameLoop, BattleEngine, Input)
├── entities/            ← 数据定义 (Parts, Tank, Bullet, Ally, Map...)
├── systems/             ← 通用系统 (CombatSystem, SkillRegistry, Synergy, Sound...)
├── skills/              ← 14 技能 (Trojan, Damocles, Noah, Mjolnir...)
├── modes/               ← 游戏模式 (Siege, TwoKings, Demo)
├── ui/                  ← 渲染 (Renderer, EffectRenderer, BattleUI, Garage...)
├── ai/                  ← AI (EnemyAI)
├── types/               ← 类型定义
└── utils/               ← 工具 (Vec2, Grid, Canvas)
```

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

打开 `http://localhost:3000` 开始游戏。
